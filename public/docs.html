üìã Overview

The Shakespeare Productions API provides access to structured data about upcoming and recent Shakespeare theatre productions worldwide. Backed by Neon PostgreSQL for accounts, API keys, usage; production lookups currently in-memory with optional DB search.

Base URL

https://shakesfind.com/api/

Data Format

All responses are in JSON format by default. XML support coming soon.

Rate Limiting

  • Free tier: 1,000 requests per day
  • Professional: 100,000 requests per month
  • Enterprise: Unlimited requests

üîê Authentication

API keys are required for production use. Include your API key in all requests:

üõ† Endpoints

Search and Lookup

GET /api/

Main search endpoint supporting multiple query types.

Parameters

Parameter Required Description Example
i Optional* Production ID prod_1
t Optional* Play title search Hamlet
s Optional* Search across play/company/venue/city Stratford
year No Filter by start year 2025
company No Filter by company substring Royal
status No Filter by status running
city No Filter by city London
country No Filter by country UK

*At least one of i, t, or s is required.

Additional Endpoints

GET /api/productions

List productions (paginated).

GET /api/years/{year}

Get all productions with start year.

GET /health

Health check endpoint.

üí° Examples

Get production by ID

GET /api/?i=prod_1&apikey=demo

Search for Hamlet 2025

GET /api/?s=Hamlet&year=2025&apikey=demo

Filter by company (Royal)

GET /api/?s=Royal&company=Royal&apikey=demo

Search by city (London)

GET /api/?s=London&city=London&apikey=demo

List productions (page 1)

GET /api/productions?page=1&pageSize=2
